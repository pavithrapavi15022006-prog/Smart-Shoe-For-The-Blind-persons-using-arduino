# 🌊 Smart Drainage Monitoring System

An Arduino-based **Smart Drainage Monitoring System** that monitors water levels in drainage channels using an ultrasonic sensor. The system provides visual and audio alerts through LEDs and a buzzer, helping detect rising water levels early and reducing the risk of overflow and flooding.

---

## 📌 Features

- 💧 Real-time water level monitoring
- 📏 Ultrasonic sensor-based distance measurement
- 🟢 Green LED for safe water level
- 🟡 Yellow LED for warning level
- 🔴 Red LED for critical water level
- 🔊 Buzzer alert during critical conditions
- ⚡ Simple, low-cost, and easy-to-build system

---

## 🛠️ Technologies Used

### Hardware
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- LEDs (Green, Yellow, Red)
- Buzzer

### Programming Language
- Embedded C

## 🔌 Circuit Connections

| Component | Arduino Pin |
|----------|-------------|
| HC-SR04 Trigger | D9 |
| HC-SR04 Echo | D10 |
| Green LED | D5 |
| Yellow LED | D6 |
| Red LED | D7 |
| Buzzer | D8 |

## 🚀 How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/YourUsername/Smart-Drainage-Monitoring-System.git
```

### 2. Open the Project

Open the `Smart_Drainage_Monitoring.ino` file in the Arduino IDE.

### 3. Connect the Hardware

Connect the ultrasonic sensor, LEDs, and buzzer to the Arduino Uno according to the circuit connections above.

### 4. Upload the Code

Select:
- **Board:** Arduino Uno
- **Port:** Your connected COM Port

Click **Upload**.
